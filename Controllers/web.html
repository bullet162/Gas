GUIDED-ADAPTIVE SMOOTHING

This system demonstrates how forecasting is improved using the Guided-Adaptive Smoothing (GAS) technique.

Upload Dataset

Accepts time series data in .CSV format for forecasting analysis.

Forecast Visualization

Generates a line chart to compare forecasted trends visually.

Accuracy Comparison

Calculates and visualizes RMSE, MSE, MAE, and MAPE across models.

How It Works

  1. Upload Dataset – (.CSV format) Contains date and value columns for time series forecasting.
  2. View Line Chart – Visualizes forecasted values using different models.
  3. View Error Metrics – Compares model accuracy using RMSE, MSE, MAE, MAPE.

The model with the lowest error metrics is considered the most accurate. GAS typically performs better due to its adaptive smoothing logic.
